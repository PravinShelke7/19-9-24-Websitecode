Section IV:

Step-by-step Procedures

This section of the User Guide provides step-by-step instructions for the functions in the Universal Manager, Global Manager, Toolbox, and Case Manager.

A.  Universal Manager

When you log in to the Savvypack® System, the first page after logging in is the Universal Manager screen.   The step-by-step procedures relevant to the Universal Manager page are:

1.  Start a module

1.  Select the desired module using the first drop down menu near the top of the screen

2.  Hit “Start Analysis” button

3.  This takes you to the Global Manager screen for the module you selected

2.  Use the Logon Renewal feature

1.  Click the Logon Renewal button.

2.  The Logon Renewal screen displays in a new window.

3.  Enter your ID and password and click the Submit button.

4.  Your security is re-set, and you can now start a module.

NOTE: This User Manual is for the SavvyPack® Packaging Analysis System.  Knowledgebases and market studies can be accessed from the Universal Manager, but instructions for these services are provided in other User Manuals.

 

B.  Global Manager

The Global Manager is the first screen that displays after selecting and starting a module on the Universal Manager screen.  The step-by-step procedures relevant to the Global Manager page are:

1.  Start a base case

1.  Select a case from the dropdown menu entitled “Select from Available Base Cases.”  

2.  Click the Start a Base Case button.

3.  The Case Manager will display in a new window.

4.  Close the Case Manager to return to the Global Manager.

2.  Start a proprietary case

1.  Select a case from the dropdown menu entitled “Select from Available Proprietary Cases.”  

2.  Click the Start a Proprietary Case button.

3.  The Case Manager will display in a new window.

4.  Close the Case Manager to return to the Global Manager.

3.  Use the Case Search feature

1.  Click the button entitled “Case Search” below the Select from Available Proprietary Cases dropdown box.

2.  The Case Search screen will display in a new window.

3.  Click the Case Viewer button to browse detailed information about each case (optional).

4.  Enter search criteria and click the Search button to find cases that match the criteria.  The matching cases are displayed in the Case Search window.

5.  Click the Case Viewer button to browse detailed information about each case (optional).

6.  Select the link for the desired case located under the column entitled “Package Format.”

7.  The Case Search window closes and the selected case is shown in the Select from Available Proprietary Cases dropdown box.

4.  Rename a case

1.  Select the case to rename from the Select from Available Proprietary Cases dropdown box.

2.  Click the Rename button located beneath the Select from Available Proprietary Cases dropdown box.

3.  Make changes to the Packaging Format and/or the Unique Features text boxes.

4.  Click the Update button to make the changes or click the Cancel button to cancel the changes.

5.  Assuming the Update button was clicked, a verification pop-up window appears and displays a message describing the action the system intends to take.

6.  Click the OK button to proceed with the action or the Cancel button to cancel the action.

7.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the case was successfully updated.

8.  Click OK to acknowledge the confirmation message and to return to the Global Manager screen.

5.  Start the Toolbox

1.  Click the Go to Toolbox button

2.  The Toolbox will display in the same window.  

3.  Click the Global Manager button to return to the Global Manager.  

6.  Logoff

1.  Click the Logoff button to logoff the system.

2.  The next screen to display is the Universal Manager.

NOTE: Always remember to logoff.  Due to security features, failing to logoff will prevent you from re-entering the system.  

C.  Toolbox

The Toolbox is selected on the Global Manager screen.  The step-by-step procedures relevant to the Toolbox page are:

1.  Copy a base case

1.  Select the source case from the dropdown box entitled “Source Base Case.”

2.  Click the Copy button directly beneath the Source Base Case dropdown box.

3.  A verification pop-up window appears and displays a message describing the action the system intends to take.

4.  Click the OK button to proceed with the action or the Cancel button to abort the action.

5.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming a new target case was created and the source case was copied successfully to the new target case.

6.  Record the new target case number (optional).

7.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

2.  Copy a proprietary case

1.  Select the source case from the dropdown box entitled “Source Proprietary Case.”

2.  Click the Copy button directly beneath the Source Proprietary Case dropdown box.

3.  A verification pop-up window appears and displays a message describing the action the system intends to take.

4.  Click the OK button to proceed with the action or the Cancel button to cancel the action.

5.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming a new target case was created and the source case was copied successfully to the new target case.

6.  Record the new target case number (optional).

7.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

3.  Transfer a base case

1.  Select the source case from the dropdown box entitled “Source Base Case.”

2.  Click the Transfer button directly beneath the Source Base Case dropdown box.

3.  Another dropdown box appears entitled “Target Proprietary Case.”

4.  Select the target case from the Target Proprietary Case dropdown.

5.  Click the Transfer button beneath the Target Proprietary Case dropdown to initiate the transfer or click the Cancel button.

6.  Assuming the Transfer button was clicked, a verification pop-up window appears and displays a message describing the action the system intends to take.

7.  Click the OK button to proceed with the action or the Cancel button to abort the action.

8.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the transfer was successfully completed.

9.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

4.  Transfer a proprietary case

1.  Select the source case from the dropdown box entitled “Source Proprietary Case.”

2.  Click the Transfer button directly beneath the Source Proprietary Case dropdown box.

3.  Another dropdown box appears entitled “Target Proprietary Case.”

4.  Select the target case from the Target Proprietary Case dropdown.

5.  Click the Transfer button beneath the Target Proprietary Case dropdown to initiate the transfer or click the Cancel button.

6.  Assuming the Transfer button was clicked, a verification pop-up window appears and displays a message describing the action the system intends to take.

7.  Click the OK button to proceed with the action or the Cancel button to abort the action.

8.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the transfer was successfully completed.

9.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

5.  Create a case

1.  Click the Create button below the Source Proprietary Case dropdown box.

2.  A verification pop-up window appears and displays a message describing the action the system intends to take.    

3.  Click the OK button to create the new case or click the Cancel button to cancel the action.

4.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the transfer was successfully completed.

5.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

6.  Delete a case

1.  Click the Delete button below the Source Proprietary Case dropdown box.

2.  A verification pop-up window appears and displays a message describing the action the system intends to take.

3.  YOU MUST BE SURE YOU WANT TO DELETE THE CASE DISPLAYED.  ONCE DELETED, THE CASE CANNOT BE RECOVERED.

4.  Click the OK button to delete the new case or click the Cancel button to cancel the delete.

5.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the delete was successfully completed.

6.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

7.  Rename a case

1.  Click the Rename button below the Source Proprietary Case dropdown box.

2.  A Rename Case Details window appears and displays the current name and description of the selected case.

3.  Enter the new name and/or description information.

4.  Click the Rename button to enter the changes or click the Cancel button to cancel the action.

5.  Assuming the Rename button was clicked, a verification window appears describing the action the system intends to take.

6.  Click the OK button to proceed with the action or click the Cancel button to cancel the action.

7.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the name and description update was successfully completed.

8.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

8.  Share access to a case

1.  Select the source case from the dropdown box entitled “Source Proprietary Case.”

2.  Click the Share Access button directly beneath the Source Proprietary Case dropdown box.

3.  Another dropdown box appears entitled “Share Access with this User.”

4.  Select the target user from the Share Access with this User dropdown.

5.  Click the Share Access button beneath the Share Access with this User dropdown to initiate the share or click the Cancel button to cancel the share.

6.  Assuming the Share Access button was clicked, a verification pop-up window appears and displays a message describing the action the system intends to take.

7.  Click the OK button to proceed with the action or click the Cancel button to abort the action.

8.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the share was successfully completed.

9.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

NOTE: Sharing access to a case means that all users are given access to the same case.  No copies have been made.  Changes made to a case by one user will change the case for every user with access to that case.

9.  Share a copy of a case

1.  Select the source case from the dropdown box entitled “Source Proprietary Case.”

2.  Click the Share a Copy button directly beneath the Source Proprietary Case dropdown box.

3.  Another dropdown box appears entitled “Share Access with this User.”

4.  Select the target user from the Share Access with this User dropdown.

5.  Click the Share a Copy button beneath the Share Access with this User dropdown to initiate the share or click the Cancel button to cancel the share.

6.  Assuming the Share a Copy button was clicked, a verification pop-up window appears and displays a message describing the action the system intends to take.

7.  Click the OK button to proceed with the action or click the Cancel button to abort the action.

8.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the share was successfully completed.

9.  Click OK to acknowledge the confirmation message and to return to the ToolBox screen.

NOTE: Sharing a copy of a case means you each user is provided a copy of the case.  As a result, changes made to a shared case by one user will NOT effect the copy of the case of other users.

10.  Synchronize a case

1.  Select the source case from the dropdown box entitled “Source Proprietary Case.”

2.  Click the Synchronize button directly beneath the Source Proprietary Case dropdown box.

3.  A verification pop-up window appears and displays a message describing the action the system intends to take.

4.  Click the OK button to proceed with the action or click the Cancel button to abort the action.

5.  Assuming the OK button was clicked, after a short delay, a confirmation pop-up window appears that displays a message confirming the synchronization was successfully completed.

6.  Click OK to acknowledge the confirmation message and to return to the Toolbox screen.

 

D.  Case Manager

The Case Manager is the first screen that is displayed after selecting a case on the Global Manager screen.  The step-by-step procedures relevant to the Case Manager page are:

1.  Access Assumptions screens

1.  Click the desired assumption screen link under the column entitled “Specify Assumptions.”  

2.  The assumption page displays in a new window.  

3.  Work in the assumption page and then close the window to return to the Case Manager.  

Note that each assumption page must be accessed and filled out correctly in order for the system to calculate results. The instructions for filling out each assumption page are provided in later sections of this manual.

2.  Access Intermediate results screens

1.  Click the link to the desired intermediate results page located in the middle column of the Case Manager screen under the column entitled “Review Intermediate Results.”  

2.  The intermediate results page displays in a new window.  

3.  Work in the intermediate results page and then close the window to return to the Case Manager.

3.  Access Final results screens

1.  Click the link to the desired results page you are interested in located in the right column of the Case Manager screen under the column entitled “Review Final Results.”  

2.  The final results page displays in a new window.  

3.  Work in the final results page and then close the window to return to the Case Manager.

NOTE: The final results pages also include a link to a compare feature that makes it possible to compare cases, both in tables and with graphs.  

4.  Access Case Preferences screen

1.  Click the link to the Case Preferences page located above the main body of the Case Manager screen.

2.  The Case Preferences page displays in a new window.  

3.  Work in the final results page and then close the window to return to the Case Manager.

NOTE: All assumptions, intermediate results, and final results pages should be closed while changing the assumptions on the Case Preferences screen.

Section VII:

Sustain1 Instructions

This section of the User Manual provides instructions for the screens of the Sustain1 Case Manager.  The instructions are presented in a sequence similar to the sequence of pages on the Case Manager.  

The input variables for each screen are defined in this section.  Typically, each input variable requires several entries arranged in one column on the input screen.  For example, up to ten materials can be selected on the Material and Structure screen, and these ten selections are positioned in one column.

 

A.  Preferences input screen

The Preferences input screen of Sustain1 is examined in the following paragraphs.  

LCI DATA SOURCE

You must select from one of the LCI data sets stored in the database.  The primary differentiators between the data sets are geographic regions and biomass energy credits.  Data sets that include biomass energy credits only include energy derived from non-renewable fossil fuels.  The available data sets include the following:

UNITEDSTATES1  

Data set designed to match scorecard values.

UNITEDSTATES2  

 

Data set based on production processes in the United States; does not include biomass energy credits.

UNITEDSTATES3  

 

Data set based on production processes in the United States; includes biomass energy credits.

EUROPE1  

 

Data set based on production processes in the Europe; does not include biomass energy credits.

EUROPE2  

 

Data set based on production processes in the Europe; includes biomass energy credits.

GLOBAL1  

 

Data set based on global average production processes; does not include biomass energy credits.

GLOBAL2  

Data set based on global average production processes; includes biomass energy credits.

 

 

CASE DATE  

You select the date for the case.  The case date sets the date of the LCI data selected earlier.

 

 

UNITS

You select the units (English or metric) for the active case.

NOTE: Close all input, intermediate results, and final results screens when changing preferences.

 

B.  Material and Structure input screen (S1)

The Material and Structure input screen of Sustain1 is examined in the following paragraphs.  

LCI DATA SOURCE

The LCI data is selected on the Preferences input screen.

 

 

INPUT VARIABLE

DEFINITIONS

 

 

PRIMARY MATERIALS

In the materials column, you select the materials used in this product. Material choices are provided from those in the materials database.

 

 

THICKNESS

You input the thickness of each material based on a cross section of the product.

 

 

RECYCLE

Certain materials, for example polymer resins, sometimes utilize recycled process waste as a raw material. You enter the percentage (0 to 100%) recycled of only in-plant waste.

 

 

EXTRA-PROCESS

Certain materials, for example used printing inks, are inventoried and reworked where possible. A portion of these materials are disposed of from inventory. You enter the percentage (0 to 100%), if any, of disposed or wasted material in addition to the waste that occurs in the production process.

 

 

ENERGY - SUGGESTED

Displays the cradle to gate energy required to produce a unit of the selected raw material.

 

 

ENERGY - PREFERRED

You may enter an energy value in the preferred column that will replace the suggested energy value in all calculations.

 

 

CO2 - SUGGESTED

Displays the cradle to gate CO2 gas equivalents generated to produce a unit of the selected raw material.

 

 

CO2 - PREFERRED

You may enter a CO2 gas equivalent value in the preferred column that will replace the suggested value in all calculations.

 

 

SHIP DISTANCE - SUGGESTED

Displays the typical shipping distance for the selected raw material.

 

 

SHIP DISTANCE - PREFERRED

You may enter a shipping distance in the preferred column that will replace the suggested value in all calculations.

 

 

RECOVERY - SUGGESTED

Displays a rating from 1 (best) to 5 (worst) that is based on the recovery rate for the selected raw material.

 

 

RECOVERY - PREFERRED

You may enter a recovery value in the preferred column that will replace the suggested recovery value in all calculations.

 

 

SUSTAINABLE MATERIAL - SUGGESTED

Displays a rating of "Preferred" or "Less Desirable" that represents the sustainability of the raw material selected.

 

 

SUSTAINABLE MATERIAL - PREFERRED

You may enter a sustainable material value in the preferred column that will replace the suggested value in all calculations.

 

 

PC RECYCLE - SUGGESTED

Displays the post consumer recycle content in decimal percent (0 to 1) for the selected raw material. This is completely different from the in-plant recycled percentage.

 

 

PC RECYCLE - PREFERRED

You may enter a post consumer value in the preferred column that will replace the suggested value in all calculations.

 

 

EXTRA-PROCESS

Certain materials, for example used printing inks, are inventoried and reworked where possible. A portion of these materials are disposed of from inventory. You enter the percentage (0 to 100%), if any, of disposed or wasted material in addition to the waste that occurs in the production process.

 

 

SPECIFIC GRAVITY - SUGGESTED

Specific gravity for each material is maintained in the database and provided here for reference.

 

 

SPECIFIC GRAVITY - PREFERRED

You may enter a specific gravity that in the preferred specific gravity column that replaces the suggested specific gravity in all calculations.

 

 

WEIGHT PER AREA

Weight per area is a calculated value included for reference.

 

 

SHIPPING UNIT WEIGHT

You MUST input the weight of the shipping unit, which equals the amount of product, typically shipped on a pallet or in a railcar.

 

 

MANUFACTURING DEPARTMENT

You select the department that purchases the material.  This selection is required.

 

 

DISCRETE MATERIALS

The discrete materials provides the ability to add a discrete component such as a fitment.

Finished equivalent production volume is the maximum production of each department normalized to the equivalent of finished product. Calculated values from intermediate screen or ratings that are inputted manually.  Fro calculated values, a lower number is better.

LOW RANGE - PREFERRED

An input number that can be used to represent a new best rating within the evaluation category

A value that represent the worst (highest expected value within the evaluation category

An input number that can be used to represent a new worst (higher value) rating within the evaluation category

 

 

WEIGHTING (%) - SUGGESTED

The weighting of the evaluation category to the final score

 

 

WEIGHTING (%) - PREFERRED

A method of adjusting the weighting of each evaluation category to the final score.  The total weighting should add up to 100.

 

 

MAX SCORE

The maximum score allowed foe the evaluation category, based on suggested or preferred weighting %.

 

 

TOTAL

Total score of each evaluation category.

AC.  Compare Cases function (S1)

Each of the results pages has a link to a Case Comparison function.  This function provides the capability to compare the results of the active case to another case.  Comparison can be made in either tabular or graphical form.  To use this function:  

1.  Click the Compare Case link.

2.  The Compare Case function displays in a new window.

3.  Choose the case you wish to compare with from the dropdown box.

4.  Choose from the comparison choices.

5.  Close the window to return to the results page.

Section V:

Econ1 Instructions

This section of the User Manual provides instructions for the screens of the Econ1 Case Manager.  The instructions are presented in a sequence similar to the sequence of pages on the Case Manager.  

The input variables for each screen are defined in this section.  Typically, each input variable requires several entries arranged in one column on the input screen.  For example, up to ten materials can be selected on the Material and Structure screen, and these ten selections are positioned in one column.

A.  Case Preferences input screen

The Case Preferences input screen of Econ1is examined in the following paragraphs.

INPUT VARIABLE

DEFINITIONS

 

 

COUNTRY OF MANUFACTURE

You select the country in which manufacturing of the product is to take place. This selection changes the suggested wages on the Personnel screen to those of the selected country.

 

 

COUNTRY DESTINATION

You select the country in which the product will be used.

 

 

UNITS

You select the units (English or metric) for the active case.

 

 

CURRENCY

You select the currency for the active case.

 

 

CASE DATE

You select the date for the case.  The case date sets the price of raw materials, salaries, and energy to the price at that date.

NOTE: Close all input, intermediate results, and final results screens when changing preferences.

 

B.  Material and Structure input screen (E1)

The Material and Structure input screen of Econ1is examined in the following paragraphs.  

INPUT VARIABLE

DEFINITIONS

 

 

PRIMARY MATERIALS

In the materials column you select the materials used in this product. Material choices are provided from those in the materials database.

 

 

THICKNESS  

You input the thickness of each material in the cross section of the product.

 

 

PRICE - SUGGESTED

The price of each material in the database is updated on a monthly basis. The suggested price is the price at the date you select in the case preferences.

 

 

PRICE - PREFERRED

You may enter a price that in the preferred price column that will replace the suggested price in all calculations. Inserting a preferred price also permanently locks that price into the model.

 

 

RECYCLE

Certain materials, for example polymer resins, sometimes utilize recycled process waste as a raw material. You enter the percentage (0 to 100%) recycled of only in-plant waste.

 

 

EXTRA-PROCESS

Certain materials, for example used printing inks, are inventoried and reworked where possible. A portion of these materials are disposed of from inventory. You enter the percentage (0 to 100%), if any, of disposed or wasted material in addition to the waste that occurs in the production process.

 

 

SPECIFIC GRAVITY - SUGGESTED

The suggested specific gravity of each material is maintained in the SavvyPack® database.

 

 

SPECIFIC GRAVITY - PREFERRED

You may enter a specific gravity that in the preferred specific gravity column that replaces the suggested specific gravity in all calculations.

 

 

WEIGHT PER AREA

Weight per area is a calculated value included for reference.

 

 

MANUFACTURING DEPARTMENT

You select the department that purchases the material.  This selection is required.

 

 

DISCRETE MATERIALS

The discrete materials provides the ability to add a discrete component such as a fitment.

 

 

PRINTING PLATES

The printing plate selector provides the ability to select the payor of printing plate cost.

C.  Product Format input screen (E1)

The Product Format input screen of Econ1 is examined in the following paragraphs.  

INPUT VARIABLE

DEFINITIONS

 

 

PRODUCT FORMAT TYPE

You select the product format from the choices maintained in the database. After selecting the product type, you must update the page. The updated screen shows the appropriate inputs for the product selected.

 

 

INPUTS

D.   Pallet and Truck Configuration input screen (E1)

The Pallet and Truck Configuration input screen of Econ1 is examined in the following paragraphs.  

INPUT VARIABLE

DEFINITIONS

 

 

PALLET VOLUME

You specify the pallet dimensions, including the allowable height, which determines the cube of volume available for the product and its packaging.

 

 

CARTONS PER PALLET

You specify the number of cartons per pallet, if applicable.

 

 

PRODUCT PER PALLET

You specify the number of products that will fit on a pallet, given the cube of volume also specified on this screen.

 

 

TRUCK VOLUME

You specify the truck dimensions, which determines the volume of truck space available for finished pallets.

 

 

TRUCK WEIGHT

You specify the truck weight limit.

 

 

PALLETS PER TRUCK

You specify the number of pallets that can be shipped in the truck based on the volume and weight limitations also specified on this screen.

Results

The truck weight is calculated and displayed.

F.  Department Configuration input screen (E1)

The Department Configuration input screen of Econ1 is examined in the following paragraphs.

INPUT VARIABLE

DEFINITIONS

 

 

REQUIRED DEPARTMENTS

You select the departments required to manufacture the product from the departments maintained in the database. For example, for a printed film you might specify an extrusion department and a printing department. Only processing departments should be specified, for example, do not select administration.

 

 

NEXT PROCESS

Several additional columns are available so you can select the sequence of processing departments that follow each department. Following the previous example, the printing department should be selected in column two immediately to the right of the extrusion department, because printing is conducted after extrusion in this example.

Q.  Operating interim results screen (E1)

The Operating interim results screen of Econ1 is examined in the following paragraphs.  

INPUT VARIABLE

DEFINITIONS

 

 

DEPARTMENT

The departments of this case are displayed.

 

 

PRODUCTION VOLUME

The maximum production volume (capacity) of each department is displayed.

 

 

FINISHED EQUIVALENT PRODUCTION VOLUME

Finished equivalent production volume is the maximum production of each department normalized to the equivalent of finished product. This calculated result is directly comparable between departments, and the lowest number is the maximum production volume. This number can also be used to re-specify process equipment to balance production between departments.

 

 

FINAL PRODUCTION VOLUME

The final production volume is the recalculated production volume, reduced to be balanced with the department that limits production.

 

 

AVERAGE DOWNTIME

The average downtime for each department is displayed.

 

 

AVERAGE WASTE

The average waste for each department is displayed.

U.  Compare Cases function (E1)

Each of the results pages has a link to a Case Comparison function.  This function provides the capability to compare the results of the active case to another case.  Comparison can be made in either tabular or graphical form.  To use this function:  

1.  Click the Compare Case link.

2.  The Compare Case function displays in a new window.

3.  Choose the case you wish to compare with from the dropdown box.

4.  Choose from the comparison choices.

5.  Close the window to return to the results page.

 

V.  Print Case Summary function (E1)

The Print Case Summary of Econ1 is examined in the following paragraphs.

Click the Print Case Summary link on the Case Manager, and a case summary is automatically generated.  It summarizes the most important results and assumptions for the following pages:

•  Profit and Loss

•  Material Assumptions

•  Materials Results

•  Equipment Assumptions

•  Operating Assumptions

•  Personnel Assumptions

•  Fixed Cost Assumptions

The generated results can be printed using your browser’s print function.

Section VIII:

Sustain2 Instructions

This section of the User Manual provides instructions for the screens of the Sustain2 Case Manager.  The instructions are presented in a sequence similar to the sequence of pages on the Case Manager.  

The input variables for each screen are defined in this section.  Typically, each input variable requires several entries arranged in one column on the input screen.  For example, up to ten materials can be selected on the Material and Structure screen, and these ten selections are positioned in one column.

 

A.  Preferences input screen

The Preferences input screen of Sustain2 is examined in the following paragraphs.   

LCI DATA SOURCE

You must select from one of the LCI data sets stored in the database. Primary differentiators are geographic regions and biomass energy credits. Data sets that include biomass energy credits will only report energy derived from non-renewable fossil fuels.

UNITEDSTATES1  

Data set designed to match scorecard values.

UNITEDSTATES2  

 

Data set based on production processes in the United States; does not include biomass energy credits.

UNITEDSTATES3  

 

Data set based on production processes in the United States; includes biomass energy credits.

EUROPE1  

 

Data set based on production processes in the Europe; does not include biomass energy credits.

EUROPE2  

 

Data set based on production processes in the Europe; includes biomass energy credits.

GLOBAL1  

 

Data set based on global average production processes; does not include biomass energy credits.

GLOBAL2  

Data set based on global average production processes; includes biomass energy credits.

 

 

CASE DATE  

You select the date for the case.  The case date sets the date of the LCI data selected earlier.

 

 

UNITS

You select the units (English or metric) for the active case.

  

NOTE: Close all input, intermediate results, and final results screens when changing preferences.

 

B.   Materials input screen (S2)

The Material and Structure input screen of Sustain2 is examined in the following paragraphs.

LCI DATA SOURCE

The LCI data is selected on the Preferences input screen.

 

 

INPUT VARIABLE

DEFINITIONS

 

 

MATERIAL SELECTION

In the materials column you select the materials used in this product. Material choices are provided from those in the materials database.

 

 

PACKAGE COMPONENT

You select the package component be included in the environmental impact analysis. Typically, this is all of the packaging materials.

 

 

QUANTITY

You input the number of units in the final product. This quantity is FOR REFERENCE ONLY.

 

 

SUSTAIN1 CASES

You select the cases in the Sustain1 module from which to draw the environmental data.

 

 

WEIGHT (LB)

 

You input the weight of each material for the total number of units in the final product.

 

 

RECYCLE (%)

 

You may enter the recycle percent (0 to 100) for the selected packaging component, if applicable.

 

 

EXTRA PROCESS (%)

 

Certain materials, for example used printing inks, are inventoried and reworked where possible. A portion of these materials are disposed of from inventory. You enter the percentage (0 to 100%), if any, of disposed or wasted material in addition to the waste that occurs in the production process.

 

 

SPECIFIC GRAVITY

Specific gravity for each material is maintained in the database and provided here for reference.

 

 

WEIGHT (%)

Weight per volume is a calculated value included for reference.

 

 

DEPARTMENT

You select the department that purchases the material.

 

 

C.  Product format input screen (S2)

The Product Format screen is used to specify the type of product and some of its characteristics. The instructions provided here are organized to correspond to the columns on the screen.

INPUT VARIABLE

DEFINITIONS

 

 

PRODUCT FORMAT TYPE

You select the product format from the choices maintained in the database. Having selected the product type, you update the page. The updated screen shows the appropriate inputs for the product selected.

 

 

INPUTS

The inputs required change with product type. For example, you specify width and height for a pouch; you specify diameter and height for a bottle.

Results

The product weight is calculated and displayed.

 

D.  Pallet and Truck Configuration input screen (S2)

The Pallet and Truck Configuration screen is used to specify the volume, weight, and product restrictions on the pallet and truck. The instructions provided here are organized to correspond to the columns on the screen.

INPUT VARIABLE

DEFINITIONS

 

 

PALLET DIMENSIONS

You specify the pallet dimensions, including the allowable height, which determines the cube of volume available for the product and its packaging.

 

 

CARTONS PER PALLET

You specify the number of cartons per pallet, if applicable.

 

 

PRODUCT PER PALLET

You specify the number of products that will fit on a pallet, given the cube of volume also specified on this screen.

 

 

TRUCK DIMENSIONS

You specify the truck dimensions, which determines the volume available for finished pallets.

 

 

TRUCK WEIGHT LIMIT

You specify the truck weight limit.

 

 

PALLETS PER TRUCK

You specify the number of pallets that can be shipped in the truck based on the volume and weight limitations also specified on this screen.

Results

The truck weight is calculated and displayed.

E.  Pallet Packaging input screen (S2)

The Pallet Packaging input screen of Sustain2 is examined in the following paragraphs.

LCI DATA SOURCE

The LCI data is stored in the database and there are no selections required.

 

 

SPECIAL ENTRIES

 

SELLING UNIT CUBE

This special entry is required if you want to generate scorecard values - it is the ratio (0 to 1) of product volume to primary package volume.

INPUT VARIABLE

DEFINITIONS

 

 

ITEM

In the item column, you select the packaging components used to package the finished product. Packaging component choices are provided from those in the database.

 

 

NUMBER

You enter the number of packaging components used to package the finished product.

 

 

NUMBER OF USES

You specify the number of times the packaging item is reused, which reduces the consumption of that item and the corresponding environmental metrics are improved.

 

 

WEIGHT EACH - SUGGESTED

The weight of each item is maintained in the database and is displayed here for reference.

 

 

WEIGHT EACH -

PREFERRED

You may enter a weight in the preferred weight column that replaces the suggested weight in all calculations.

 

 

ENERGY - SUGGESTED

Displays the cradle to gate energy required to produce a unit of the selected packaging component.

 

 

ENERGY - PREFERRED

You may enter an energy value in the preferred column that will replace the suggested value in all calculations.

 

 

CO2 EQUIVALENT - SUGGESTED

Displays the cradle to gate CO2 gas equivalents generated to produce a unit of the selected packaging component.

 

 

CO2 EQUIVALENT - PREFERRED

You may enter a CO2 gas equivalent value in the preferred column that will replace the suggested value in all calculations

 

 

RECOVERY - SUGGESTED

Displays a rating from 1 (best) to 5 (worst) that is based on the recovery rate for the selected raw material.

 

 

RECOVERY - PREFERRED

You may enter a recovery value in the preferred column that will replace the suggested recovery value in all calculations.

 

 

SUSTAINABLE MATERIALS - SUGGESTED

Displays a rating of "Nothing," "Preferred,"  "Less Desirable," or "Not Desirable" that represents the sustainability of the raw material selected.

 

 

SUSTAINABLE MATERIALS - PREFFERED

You may enter a sustainable material value in the preferred column that will replace the suggested value in all calculations.

 

 

PC RECYCLE - SUGGESTED

Displays the post consumer recycle content in decimal percent (0 to 1) for the selected raw material. This is completely different from the in-plant recycled percentage.

 

 

PC RECYCLE - PREFERRED

You may enter a post consumer recycle value in the preferred column that will replace the suggested value in all calculations.

 

 

SHIP - SUGGESTED

Displays the typical shipping distance for the selected packaging component

 

 

SHIP - PREFERRED

 

You may enter a shipping distance in the preferred column that will replace the suggested value in all calculations.

 

 

DEPARTMENT

You specify the department that is responsible for the cost of these items.  This selection is optional.

The Operating intermediated results screen of Sustain2 displays a summary of the operating parameters for each department. The instructions provided here are organized to correspond to the columns on the screen.

INPUT VARIABLE

DEFINITIONS

 

 

DEPARTMENT

The departments of this case are displayed.

 

 

PRODUCTION VOLUME

Displays the maximum production volume (capacity) of each department.

 

 

FINISHED EQUIVALENT PRODUCTION VOLUME

Finished equivalent production volume is the maximum production of each department normalized to the equivalent of finished product. This calculated result is directly comparable between departments, and the lowest number is the maximum production volume. This number can also be used to re-specify process equipment to balance production between departments.

Displays the average waste for each department.

The scorecard results screen displays the final disposition of the materials flowing through the production facility. The instructions provided here are organized to correspond to the headings on the screen.

INPUT VARIABLE

DEFINITIONS

TYPE

A descriptive of each scorecard derivative generated using the SavvyPack® System

 

 

RAW SCORE

Calculated values from intermediate screen or ratings that are inputted manually.  For calculated values, a lower number is better.

 

 

LOW RANGE - SUGGESTED

A value that is considered better (lower) for the evaluation category.

 

 

LOW RANGE - PREFERRED

An input number that can be used to represent a new best rating within the evaluation category

 

 

HIGH RANGE - SUGGESTED

A value that represent the worst (highest expected value within the evaluation category

 

 

HIGH RANGE - PREFERRED

An input number that can be used to represent a new worst (higher value) rating within the evaluation category

 

 

 

WEIGHTING (%) - SUGGESTED

The weighting of the evaluation category to the final score

 

 

 

WEIGHTING (%) - PREFERRED

A method of adjusting the weighting of each evaluation category to the final score.  The total weighting should add up to 100.

 

 

MAX SCORE

The maximum score allowed foe the evaluation category, based on suggested or preferred weighting %.

 

 

 

TOTAL

Total score of each evaluation category.

 

 

FIRST COLUMN - ROW DEFINITIONS

 

 

 

PURCHASED MATERIAL GHG

A measurement of the GHG from materials used for product packaging and transportation.

 

 

SUSTAINABLE MATERIAL

A rating of the abundance material used

 

 

TRANSPORTATION DISTANCE

A rating based on the distance the package component will be transported to the product packaging plant

 

 

PACKAGE TO PRODUCT RATIO

A measure of weight of packaging material used to the weight of product that is being packaged  This does not have a value in S1

 

 

CUBE UTILIZATION

A measurement related to the volume of unused space in a transported pallet.

 

 

RECYCLED CONTENT

A measurement of the ratio of non-recycled material used in the packaging component

 

 

RECOVERY

A rating of how well the packaging materials used can be recovered for recycling

 

 

RENEWABLE ENERGY

A rating given to suppliers to recognized the use of renewable energy at a level above the national average.

 

 

ENERGY INNOVATION

A rating given to recognize suppliers for sustainability improvements in processes and products.
